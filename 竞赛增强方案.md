## 📖 核心概念解析:什么是四分类?

### ❓ 为什么需要四分类?

**当前二分类的局限性**:

你现在的系统只能判断"正常"或"圆锥角膜"两种状态,这在临床应用中存在明显不足:

```
当前二分类 (简单但不够):
┌─────────┬──────────┐
│  正常   │ 圆锥角膜  │
│ (250例) │  (250例) │
└─────────┴──────────┘
问题: 只知道"有病"或"没病",不知道病情严重程度!
```

**临床现实需求**:

圆锥角膜是一个**渐进性疾病**,从早期到晚期有明显的阶段性变化:

```
疾病进展过程:
正常 → 早期(轻度) → 中期(中度) → 晚期(重度)
 │        │            │            │
 │        │            │            └─ 需角膜移植
 │        │            └─ 可做交联术,禁忌激光手术
 │        └─ 可密切观察,谨慎考虑激光手术
 └─ 可安全进行激光手术
```

**不同阶段的治疗方案完全不同**:

| 阶段 | 临床特征 | 治疗方案 | 能否激光手术 |
|------|----------|----------|--------------|
| 正常 | 角膜形态正常 | 定期体检即可 | ✅ 可以 |
| 轻度KC | K值轻微升高,无明显症状 | 观察+随访 | ⚠️ 需谨慎评估 |
| 中度KC | K值明显升高,视力下降 | 角膜交联术 | ❌ 禁忌 |
| 重度KC | 角膜严重变薄、变形 | 需角膜移植 | 🚫 绝对禁忌 |

**二分类的问题**:

如果只告诉医生"这是圆锥角膜",医生会问:
- ❓ "轻度还是重度?"
- ❓ "能做激光手术吗?"
- ❓ "需要怎么治疗?"
- ❓ "会不会继续恶化?"

**二分类无法回答这些问题!**

### 3.1 方案一: 多模型集成学习

```python
# 集成策略: Voting + Stacking
class EnsembleModel:
    def __init__(self):
        # 模型1: ConvNeXt V2 (局部特征)
        self.model1 = create_model('convnextv2_base', num_classes=4)
        
        # 模型2: Swin Transformer V2 (全局特征)
        self.model2 = create_model('swinv2_base_window16_256', num_classes=4)
        
        # 模型3: EfficientNet V2 (轻量级)
        self.model3 = create_model('tf_efficientnetv2_m', num_classes=4)
        
        # 元学习器 (Stacking)
        self.meta_learner = nn.Linear(12, 4)  # 3 models * 4 classes
    
    def forward(self, x):
        # 获取各模型预测概率
        p1 = F.softmax(self.model1(x), dim=1)
        p2 = F.softmax(self.model2(x), dim=1)
        p3 = F.softmax(self.model3(x), dim=1)
        
        # 拼接后输入元学习器
        ensemble_features = torch.cat([p1, p2, p3], dim=1)
        output = self.meta_learner(ensemble_features)
        return output
```

### 3.2 方案二: 注意力机制增强

#### 技术方案

```python
# CBAM注意力模块
class CBAM(nn.Module):
    def __init__(self, channels, reduction=16):
        super().__init__()
        # 通道注意力
        self.channel_attention = nn.Sequential(
            nn.AdaptiveAvgPool2d(1),
            nn.Linear(channels, channels // reduction),
            nn.ReLU(),
            nn.Linear(channels // reduction, channels),
            nn.Sigmoid()
        )
        # 空间注意力
        self.spatial_attention = nn.Sequential(
            nn.Conv2d(2, 1, kernel_size=7, padding=3),
            nn.Sigmoid()
        )
    
    def forward(self, x):
        # 通道注意力
        ca = self.channel_attention(x)
        x = x * ca.unsqueeze(-1).unsqueeze(-1)
        
        # 空间注意力
        avg_out = torch.mean(x, dim=1, keepdim=True)
        max_out, _ = torch.max(x, dim=1, keepdim=True)
        sa = self.spatial_attention(torch.cat([avg_out, max_out], dim=1))
        x = x * sa
        
        return x

# 在ConvNeXt中插入CBAM
class ConvNeXtWithCBAM(nn.Module):
    def __init__(self, base_model):
        super().__init__()
        self.features = base_model.features
        self.cbam = CBAM(channels=1024)  # ConvNeXt Base最后一层通道数
        self.head = base_model.head
    
    def forward(self, x):
        x = self.features(x)
        x = self.cbam(x)  # 添加注意力
        x = self.head(x)
        return x
```

#### 优势

- ✅ 提升对关键区域的关注
- ✅ 为可解释性提供基础
- ✅ 技术创新点明确
- ✅ 不增加太多参数

#### 实施难度

⭐⭐ (较易,在现有模型基础上改进)

### 3.3 方案三: 多任务学习

#### 技术方案

同时进行**分类 + 分割 + 检测**三个任务:

```python
class MultiTaskModel(nn.Module):
    def __init__(self, backbone='convnextv2_base'):
        super().__init__()
        # 共享backbone
        self.backbone = create_model(backbone, features_only=True, out_indices=[3])
        
        # 任务1: 分类头 (正常/轻度/中度/重度)
        self.classifier = nn.Sequential(
            nn.AdaptiveAvgPool2d(1),
            nn.Flatten(),
            nn.Linear(1024, 4)
        )
        
        # 任务2: 分割头 (角膜区域分割)
        self.segmenter = nn.Sequential(
            nn.Conv2d(1024, 512, 3, padding=1),
            nn.Upsample(scale_factor=16),
            nn.Conv2d(512, 1, 1),
            nn.Sigmoid()
        )
        
        # 任务3: 异常区域检测
        self.detector = nn.Sequential(
            nn.Conv2d(1024, 256, 3, padding=1),
            nn.AdaptiveAvgPool2d(7),
            nn.Flatten(),
            nn.Linear(256 * 7 * 7, 4)  # [x, y, w, h]
        )
    
    def forward(self, x):
        features = self.backbone(x)[0]  # 最后一层特征
        
        # 多任务输出
        cls_output = self.classifier(features)
        seg_output = self.segmenter(features)
        det_output = self.detector(features)
        
        return {
            'classification': cls_output,
            'segmentation': seg_output,
            'detection': det_output
        }
```

#### 优势

- ✅ 任务多样性,展示技术广度
- ✅ 多任务互相促进,提升性能
- ✅ 临床价值更高(定位+分割)
- ✅ 竞赛创新亮点

```python
# SimCLR自监督学习
class SimCLR(nn.Module):
    def __init__(self, encoder, projection_dim=128):
        super().__init__()
        self.encoder = encoder
        self.projection = nn.Sequential(
            nn.Linear(1024, 512),
            nn.ReLU(),
            nn.Linear(512, projection_dim)
        )
    
    def forward(self, x1, x2):
        # 两个增强视图
        h1 = self.encoder(x1)
        h2 = self.encoder(x2)
        
        z1 = self.projection(h1)
        z2 = self.projection(h2)
        
        # NT-Xent损失
        loss = nt_xent_loss(z1, z2, temperature=0.5)
        return loss
```

```python
class GradCAM:
    def __init__(self, model, target_layer):
        self.model = model
        self.target_layer = target_layer
        self.gradients = None
        self.activations = None
        
        # 注册钩子
        target_layer.register_forward_hook(self.save_activation)
        target_layer.register_backward_hook(self.save_gradient)
    
    def save_activation(self, module, input, output):
        self.activations = output
    
    def save_gradient(self, module, grad_input, grad_output):
        self.gradients = grad_output[0]
    
    def generate_cam(self, x, target_class):
        # 前向传播
        output = self.model(x)
        
        # 反向传播
        self.model.zero_grad()
        output[0, target_class].backward()
        
        # 计算权重
        weights = torch.mean(self.gradients, dim=[2, 3], keepdim=True)
        cam = torch.sum(weights * self.activations, dim=1, keepdim=True)
        cam = F.relu(cam)
        
        # 归一化并上采样
        cam = F.interpolate(cam, size=x.shape[2:], mode='bilinear')
        cam = (cam - cam.min()) / (cam.max() - cam.min())
        
        return cam

# 使用示例
```

```python
class DetailedHeatmapAnalysis:
    """详细热力图分析,展示模型关注的区域"""
    
    def analyze_attention_regions(self, heatmap, original_image):
        """分析模型关注的区域"""
        # 提取高关注区域
        threshold = 0.7
        high_attention_mask = heatmap > threshold
        
        # 连通区域分析
        labeled_array, num_features = ndimage.label(high_attention_mask)
        
        regions = []
        for i in range(1, num_features + 1):
            region_mask = (labeled_array == i)
            region_coords = np.where(region_mask)
            
            # 计算区域特征
            region_info = {
                'id': i,
                '中心位置': self._get_centroid(region_coords),
                '面积占比': f"{np.sum(region_mask) / region_mask.size * 100:.1f}%",
                '平均关注度': f"{np.mean(heatmap[region_mask]):.2f}",
                '区域类型': self._classify_region_type(region_coords, original_image.shape)
            }
            regions.append(region_info)
        
        return sorted(regions, key=lambda x: x['平均关注度'], reverse=True)
    
    def _classify_region_type(self, coords, shape):
        """判断区域类型"""
        center_y, center_x = shape[0] // 2, shape[1] // 2
        region_center_y = np.mean(coords[0])
        region_center_x = np.mean(coords[1])
        
        # 判断位置
        if abs(region_center_y - center_y) < shape[0] * 0.2:
            if abs(region_center_x - center_x) < shape[1] * 0.2:
                return "中央角膜区域"
            else:
                return "周边角膜区域"
        elif region_center_y > center_y:
            return "下方角膜区域"
        else:
            return "上方角膜区域"
```

```python
class SimilarCaseRetriever:
    """相似病例检索"""
    
    def find_similar_cases(self, current_features, database, top_k=5):
        """从数据库中检索相似病例"""
        similar_cases = []
        
        for case in database:
            similarity = self._calculate_similarity(
                current_features, 
                case['features']
            )
            similar_cases.append({
                'case_id': case['id'],
                'similarity': similarity,
                'diagnosis': case['diagnosis'],
                'treatment': case['treatment'],
                'outcome': case['outcome']
            })
        
        # 按相似度排序
        return sorted(similar_cases, key=lambda x: x['similarity'], reverse=True)[:top_k]
    
    def _calculate_similarity(self, features1, features2):
        """计算特征相似度"""
        # 使用余弦相似度或其他度量
        vec1 = self._features_to_vector(features1)
        vec2 = self._features_to_vector(features2)
        
        similarity = cosine_similarity([vec1], [vec2])[0][0]
        return similarity
```

```python
def mixup_data(x, y, alpha=0.2):
    """MixUp增强"""
    lam = np.random.beta(alpha, alpha)
    batch_size = x.size(0)
    index = torch.randperm(batch_size)
    
    mixed_x = lam * x + (1 - lam) * x[index]
    y_a, y_b = y, y[index]
    
    return mixed_x, y_a, y_b, lam

def mixup_criterion(criterion, pred, y_a, y_b, lam):
    """MixUp损失"""
    return lam * criterion(pred, y_a) + (1 - lam) * criterion(pred, y_b)
```

```python
class DistillationTrainer:
    def __init__(self, teacher_model, student_model, temperature=4.0, alpha=0.7):
        self.teacher = teacher_model
        self.student = student_model
        self.temperature = temperature
        self.alpha = alpha  # 蒸馏损失权重
    
    def distillation_loss(self, student_output, teacher_output, labels):
        # 软标签损失
        soft_loss = F.kl_div(
            F.log_softmax(student_output / self.temperature, dim=1),
            F.softmax(teacher_output / self.temperature, dim=1),
            reduction='batchmean'
        ) * (self.temperature ** 2)
        
        # 硬标签损失
        hard_loss = F.cross_entropy(student_output, labels)
        
        # 组合损失
        return self.alpha * soft_loss + (1 - self.alpha) * hard_loss
```

#### 5.1.1 智能术前评估

```python
class PreoperativeAssessment:
    """术前评估系统"""
    
    def __init__(self):
        self.classifier = ModelService().model
        self.thresholds = {
            'k_max': 48.0,  # K值上限
            'k_min': 40.0,  # K值下限
            'thickness': 500,  # 角膜厚度下限(μm)
            'is_value': 1.8   # I-S值上限
        }
    
    def assess(self, image, clinical_data):
        """
        综合评估
        
        参数:
            image: 角膜地形图
            clinical_data: 临床数据 (K值、厚度、I-S值等)
        """
        # AI诊断
        ai_result = self.classifier.predict(image)
        
        # 规则评估
        rule_result = self._rule_based_assessment(clinical_data)
        
        # 综合决策
        final_decision = self._make_decision(ai_result, rule_result)
        
        return {
            'ai_diagnosis': ai_result,
            'clinical_assessment': rule_result,
            'final_decision': final_decision['decision'],
            'risk_factors': final_decision['risk_factors'],
            'surgery_recommendation': final_decision['recommendation']
        }
    
    def _rule_based_assessment(self, data):
        """基于临床规则的评估"""
        issues = []
        
        # K值评估
        if data['k_max'] > self.thresholds['k_max']:
            issues.append(f"K值偏高 ({data['k_max']:.1f}D > {self.thresholds['k_max']}D)")
        
        # 角膜厚度评估
        if data['thickness'] < self.thresholds['thickness']:
            issues.append(f"角膜偏薄 ({data['thickness']}μm < {self.thresholds['thickness']}μm)")
        
        # I-S值评估
        if data['is_value'] > self.thresholds['is_value']:
            issues.append(f"I-S值异常 ({data['is_value']:.1f}D > {self.thresholds['is_value']}D)")
        
        return {
            'issues': issues,
            'pass': len(issues) == 0
        }
    
    def _make_decision(self, ai_result, rule_result):
        """综合决策"""
        risk_factors = []
        
        # AI诊断为异常
        if ai_result['prediction'] != 'Normal':
            risk_factors.append(f"AI诊断: {ai_result['class_name']}")
        
        # 临床规则不通过
        if not rule_result['pass']:
            risk_factors.extend(rule_result['issues'])
        
        # 决策逻辑
```

```python
class ProgressionPredictor:
    """圆锥角膜进展预测"""
    
    def __init__(self):
        # 使用LSTM/Transformer处理时序数据
        self.model = nn.LSTM(input_size=512, hidden_size=256, num_layers=2, batch_first=True)
        self.classifier = nn.Linear(256, 3)  # 稳定/缓慢进展/快速进展
    
    def predict_progression(self, historical_images):
        """
        基于历史检查预测疾病进展
        
        参数:
            historical_images: 历史角膜地形图序列 (N张图片)
        """
        # 提取每张图片的特征
        features = []
        for img in historical_images:
            feat = self._extract_features(img)
            features.append(feat)
        
        # 构建时序输入
        sequence = torch.stack(features).unsqueeze(0)  # [1, N, 512]
        
        # LSTM预测
        lstm_out, _ = self.model(sequence)
        prediction = self.classifier(lstm_out[:, -1, :])  # 取最后时刻
        
        return {
            'progression_status': ['稳定', '缓慢进展', '快速进展'][prediction.argmax()],
            'risk_score': torch.softmax(prediction, dim=1).max().item(),
            'recommendation': self._generate_recommendation(prediction)
        }
```

```python
class MultiModalFusion(nn.Module):
    """多模态融合模型"""
    
    def __init__(self):
        super().__init__()
        
        # 模态1: 角膜地形图
        self.topography_encoder = create_model('convnextv2_base', features_only=True)
        
        # 模态2: OCT图像
        self.oct_encoder = create_model('resnet50', features_only=True)
        
        # 模态3: 临床数值数据
        self.numeric_encoder = nn.Sequential(
            nn.Linear(10, 64),  # K值、厚度、I-S值等
            nn.ReLU(),
            nn.Linear(64, 128)
        )
        
        # 融合层
        self.fusion = nn.Sequential(
            nn.Linear(2048 + 2048 + 128, 512),
            nn.ReLU(),
            nn.Dropout(0.3),
            nn.Linear(512, 4)  # 分类
        )
    
    def forward(self, topography, oct_image, numeric_data):
        # 提取各模态特征
        topo_feat = self.topography_encoder(topography)[-1].flatten(1)
        oct_feat = self.oct_encoder(oct_image)[-1].flatten(1)
        num_feat = self.numeric_encoder(numeric_data)
        
        # 拼接融合
        fused = torch.cat([topo_feat, oct_feat, num_feat], dim=1)
        
        # 分类
        output = self.fusion(fused)
        return output
```
