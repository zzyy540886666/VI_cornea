# 角膜地形图分类系统 — 优化方案

> 基于当前项目状态（ConvNeXt V2 Base 单图二分类 + Streamlit/FastAPI/Docker），制定以下分阶段优化计划。
> 目标：在有限数据量（~400 样本）下，最大化模型的**说服力**和**系统完整性**。

## 一、训练流程优化

### 1.1 差异化学习率与冻结微调（Frozen Backbone Warm-up）

**问题**：当前所有参数统一学习率 1e-4，预训练主干和随机初始化的分类头以相同速率更新，容易破坏已学到的特征。

**方案**：先冻结主干只训练分类头（warm-up），再解冻全网络以差异学习率微调。

```python
# 阶段1：冻结主干，只训练分类头（5 epoch）
for name, param in model.named_parameters():
    if 'head' not in name:          # ConvNeXt V2 的分类头层名含 'head'
        param.requires_grad = False
optimizer_head = AdamW(filter(lambda p: p.requires_grad, model.parameters()), lr=1e-3, weight_decay=0.05)

# 阶段2：解冻全网络，差异学习率（25 epoch）
for param in model.parameters():
    param.requires_grad = True
optimizer_full = AdamW([
    {'params': model.head.parameters(),          'lr': 1e-3},
    {'params': model.stages.parameters(),         'lr': 1e-4},
    {'params': model.stem.parameters(),           'lr': 1e-4},
], weight_decay=0.05)
```

**预期效果**：减少初期特征破坏，提升收敛稳定性。

### 1.2 Focal Loss 处理类别不平衡

**问题**：当前使用标准 `CrossEntropyLoss`，如果正常/异常比例不均衡（如 7:3），模型会偏向多数类。

**方案**：替换为 Focal Loss，自动降低易分类样本的权重，聚焦难分类样本。

```python
class FocalLoss(nn.Module):
    def __init__(self, alpha=None, gamma=2.0):
        super().__init__()
        self.alpha = alpha        # 类别权重，如 [0.7, 0.3]
        self.gamma = gamma        # 聚焦参数，越大越关注难样本

    def forward(self, inputs, targets):
        ce_loss = F.cross_entropy(inputs, targets, reduction='none', weight=self.alpha)
        pt = torch.exp(-ce_loss)
        focal_loss = ((1 - pt) ** self.gamma) * ce_loss
        return focal_loss.mean()

# 使用：根据实际类别比例计算 alpha
# 如果 normal:kc = 7:3，则 alpha = [0.3, 0.7]（少数类权重更高）
criterion = FocalLoss(alpha=torch.tensor([w_kc, w_normal]).to(device), gamma=2.0)
```

**预期效果**：提升对圆锥角膜（少数类）的召回率（Sensitivity），减少漏诊。

---

### 1.3 Mixup / CutMix 数据增强

**问题**：400 样本量偏小，当前增强策略（翻转、颜色抖动）较为基础。

**方案**：在训练中加入 Mixup（图像混合）或 CutMix（图像裁剪粘贴），有效扩充训练分布。

```python
# 在 train_epoch 中，前向传播前加入：
if np.random.random() < 0.3:  # 30% 概率触发
    lam = np.random.beta(0.2, 0.2)
    rand_index = torch.randperm(images.size(0)).to(device)
    images = lam * images + (1 - lam) * images[rand_index]
    targets = lam * targets + (1 - lam) * targets[rand_index]
```

**预期效果**：提升模型泛化能力，降低过拟合风险。

### 1.4 自动计算类别权重

**问题**：即使不用 Focal Loss，CrossEntropyLoss 也应传入 `weight` 参数。

**方案**：在 `load_data()` 后自动计算权重。

```python
from sklearn.utils.class_weight import compute_class_weight

# 收集所有训练标签
train_labels = [label for _, label in train_dataset.samples]
class_weights = compute_class_weight('balanced', classes=[0, 1], y=train_labels)
class_weights = torch.tensor(class_weights, dtype=torch.float32).to(self.device)
self.criterion = nn.CrossEntropyLoss(weight=class_weights)
```

## 二、评估体系完善

### 2.1 补充 AUC-ROC 和 PR 曲线

**问题**：当前只有 Accuracy / Precision / Recall / F1，缺少医学影像领域最看重的 AUC。

**方案**：在 `evaluate_test()` 中添加 ROC 和 PR 曲线绘制。

```python
from sklearn.metrics import roc_curve, auc, precision_recall_curve, average_precision_score

def plot_roc_pr(y_true, y_prob, save_dir):
    fig, (ax1, ax2) = plt.subplots(1, 2, figsize=(14, 5))

    # ROC 曲线
    fpr, tpr, _ = roc_curve(y_true, y_prob)
    roc_auc = auc(fpr, tpr)
    ax1.plot(fpr, tpr, color='#588157', lw=2, label=f'AUC = {roc_auc:.4f}')
    ax1.plot([0, 1], [0, 1], 'k--', lw=1)
    ax1.set_xlabel('False Positive Rate')
    ax1.set_ylabel('True Positive Rate')
    ax1.set_title('ROC Curve')
    ax1.legend()
    ax1.grid(True, alpha=0.3)

    # PR 曲线
    precision, recall, _ = precision_recall_curve(y_true, y_prob)
    ap = average_precision_score(y_true, y_prob)
    ax2.plot(recall, precision, color='#BC4749', lw=2, label=f'AP = {ap:.4f}')
    ax2.set_xlabel('Recall')
    ax2.set_ylabel('Precision')
    ax2.set_title('Precision-Recall Curve')
    ax2.legend()
    ax2.grid(True, alpha=0.3)

    plt.tight_layout()
    plt.savefig(save_dir / 'roc_pr_curves.png', dpi=150)
```

**注意**：需要修改 `validate()` 方法，返回概率值（而非只返回类别预测）。

## 三、可解释性：Grad-CAM 热力图

### 3.1 集成 Grad-CAM

**问题**：模型是"黑盒"，无法解释决策依据。评委最看重这一点。

**方案**：使用 `pytorch-grad-cam` 库，生成热力图叠加在原始地形图上。

```bash
pip install grad-cam
```

```python
from pytorch_grad_cam import GradCAM
from pytorch_grad_cam.utils.image import show_cam_on_image
from pytorch_grad_cam.utils.model_targets import ClassifierOutputTarget

def generate_gradcam(model, image_tensor, target_class, device):
    """
    生成 Grad-CAM 热力图

    参数:
        model: 训练好的模型
        image_tensor: [1, 3, 224, 224] 预处理后的张量
        target_class: 目标类别索引 (0=KC, 1=Normal)
        device: 计算设备
    """
    # 使用最后一个卷积块作为目标层
    # ConvNeXt V2 的特征提取层结构需要确认
    target_layers = [model.stages[-1].blocks[-1]]  # 最后一个 stage 的最后一个 block

    cam = GradCAM(model=model, target_layers=target_layers, use_cuda=(device.type == 'cuda'))

    targets = [ClassifierOutputTarget(target_class)]
    grayscale_cam = cam(input_tensor=image_tensor, targets=targets)
    grayscale_cam = grayscale_cam[0]  # [H, W]

    return grayscale_cam

def overlay_heatmap(original_image, grayscale_cam, alpha=0.4):
    """
    将热力图叠加到原始图像上

    参数:
        original_image: PIL Image
        grayscale_cam: [H, W] numpy array (0~1)
        alpha: 热力图透明度
    """
    import cv2
    img = np.array(original_image).astype(np.float32) / 255.0
    img_resized = cv2.resize(img, (grayscale_cam.shape[1], grayscale_cam.shape[0]))
    heatmap = show_cam_on_image(img_resized, grayscale_cam, use_rgb=True, image_weight=1 - alpha)
    return heatmap
```

### 3.3 API 接口支持热力图

**方案**：在 FastAPI 的 `/predict` 接口中，增加 `with_cam` 参数，返回 base64 编码的热力图。

```python
@app.post("/predict")
async def predict(image: UploadFile, with_cam: bool = False):
    # ... 原有预测逻辑 ...

    if with_cam:
        cam_image = generate_gradcam(...)  # 返回 numpy array
        _, buffer = cv2.imencode('.png', cam_image)
        cam_base64 = base64.b64encode(buffer).decode('utf-8')
        result['heatmap'] = f"data:image/png;base64,{cam_base64}"

    return result
```

## 四、数据分布可视化

### 4.1 数据集概览脚本

**方案**：新建 `analyze_data.py`，自动分析并输出数据集统计图。

```python
"""
数据集分析与可视化
生成数据分布报告，用于答辩材料和文档补充
"""

def analyze_dataset(data_dir='data'):
    stats = {
        'normal': len(list(Path(data_dir, 'normal').glob('*.jpg')) +
                     list(Path(data_dir, 'normal').glob('*.png'))),
        'kc': len(list(Path(data_dir, 'kc').glob('*.jpg')) +
                  list(Path(data_dir, 'kc').glob('*.png'))),
    }

    fig, axes = plt.subplots(1, 2, figsize=(12, 5))

    # 饼图：类别比例
    axes[0].pie([stats['normal'], stats['kc']],
                labels=['正常角膜', '圆锥角膜'],
                colors=['#588157', '#BC4749'],
                autopct='%1.1f%%', startangle=90)
    axes[0].set_title('类别分布')

    # 柱状图：各类别样本量
    bars = axes[1].bar(['正常角膜', '圆锥角膜'], [stats['normal'], stats['kc']],
                       color=['#588157', '#BC4749'])
    axes[1].set_ylabel('样本数')
    axes[1].set_title('各类别样本量')
    for bar, val in zip(bars, [stats['normal'], stats['kc']]):
        axes[1].text(bar.get_x() + bar.get_width()/2, bar.get_height() + 2,
                     str(val), ha='center', fontweight='bold')

    plt.tight_layout()
    plt.savefig('checkpoints/data_distribution.png', dpi=150)
```
