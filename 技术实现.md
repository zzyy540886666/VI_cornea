## 一、项目概述

本项目基于深度学习技术，实现角膜地形图的自动二分类（正常角膜 / 圆锥角膜），辅助近视激光手术术前评估。

### 1.1 技术栈

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| 深度学习框架 | PyTorch 2.4+ | 动态图框架，灵活高效 |
| 模型库 | timm 1.0+ | PyTorch Image Models，提供最新预训练模型 |
| Web 界面 | Streamlit 1.28+ | Python 原生 Web 框架，快速搭建可视化界面 |
| API 服务 | FastAPI 0.104+ | 高性能异步 Web 框架，自动生成 API 文档 |
| ASGI 服务器 | Uvicorn 0.24+ | 异步 ASGI 服务器 |
| 数据处理 | PIL、OpenCV、scikit-learn | 图像读取、预处理、评估指标 |
| 可视化 | Matplotlib | 训练曲线、混淆矩阵、预测结果可视化 |
| 容器化 | Docker + Docker Compose | 生产环境部署 |

## 二、数据处理

### 2.1 数据集结构

原始数据存放在 `data/` 目录，按类别分子文件夹：

```
data/
├── kc/          # 圆锥角膜（异常）图片
└── normal/      # 正常角膜图片
```

支持的图片格式：`.jpg`、`.png`。

### 2.2 数据划分

通过 `prepare_data.py` 将数据按 **70% / 15% / 15%** 的比例划分为训练集、验证集、测试集。

**划分策略：**
- 使用 `random.seed(42)` 固定随机种子，保证划分结果可复现
- 每个类别独立随机打乱后按比例切分
- 使用 `shutil.copy2` 复制文件到目标目录，保留原始数据不变

**实现要点：**

```python
# prepare_data.py 核心逻辑
random.seed(42)
for class_name in ['kc', 'normal']:
    images = list(class_dir.glob("*.jpg")) + list(class_dir.glob("*.png"))
    random.shuffle(images)
    
    total = len(images)
    train_end = int(total * 0.70)      # 70% 训练集
    val_end = int(total * 0.85)        # 15% 验证集 + 70% 训练集
    
    splits = {
        'train': images[:train_end],
        'val': images[train_end:val_end],
        'test': images[val_end:]
    }
```

### 2.4 数据增强

训练阶段与验证阶段使用不同的数据预处理策略：

**训练增强（train_transform）：**

| 增强方法 | 参数 | 作用 |
|---------|------|------|
| RandomResizedCrop | scale=(0.8, 1.0), size=224 | 随机缩放裁剪到 224x224，模拟不同拍摄距离 |
| RandomHorizontalFlip | p=0.5 | 水平翻转，模拟镜像 |
| RandomVerticalFlip | p=0.5 | 垂直翻转，角膜地形图上下对称 |
| ColorJitter | brightness=0.2, contrast=0.2, saturation=0.2 | 颜色抖动，增强对光照变化的鲁棒性 |
| Normalize | ImageNet 均值/标准差 | 标准化到 [-1, 1] 范围，匹配预训练权重 |

**验证/测试预处理（val_transform）：**

| 步骤 | 参数 | 作用 |
|------|------|------|
| Resize | 224 | 等比缩放短边到 224 |
| CenterCrop | 224 | 中心裁剪到 224x224 |
| Normalize | ImageNet 均值/标准差 | 标准化 |

**归一化参数：**
```python
mean = [0.485, 0.456, 0.406]   # ImageNet 数据集 RGB 均值

```

## 三、模型设计

### 3.1 模型选型：ConvNeXt V2 Base

**选择理由：**

1. **卷积局部特征提取**：角膜地形图具有丰富的局部纹理信息（环状色彩分布），卷积操作天然适合捕获此类局部模式
2. **全局响应归一化（GRN）**：ConvNeXt V2 独有的 GRN 层通过全局竞争机制增强关键特征，抑制噪声
3. **FCMAE 预训练**：在 ImageNet-22K 上使用全卷积掩码自编码器（FCMAE）预训练，学习更丰富的特征表示
4. **迁移学习友好**：在医学影像小样本场景下，预训练权重的迁移效果显著优于从头训练

**模型参数：**

| 参数 | 值 |
|------|-----|
| 模型名称 | `convnextv2_base` |
| 预训练数据集 | ImageNet-22K（1400 万张图片）|
| 预训练方式 | FCMAE (Fully Convolutional Masked Autoencoder) |
| 输入尺寸 | 224 x 224 x 3 |
| 参数量 | 88,591,394（约 88M） |
| 分类头 | 替换为 2 类全连接层 |

### 3.2 模型构建

使用 timm 库的 `create_model` 接口加载模型：

```python
from timm import create_model

model = create_model(
    'convnextv2_base',    # 模型标识符
    pretrained=True,       # 加载 ImageNet 预训练权重
    num_classes=2,         # 替换分类头为 2 类
    in_chans=3             # 输入 3 通道 RGB
)
```

**加载流程：**
1. timm 根据 `pretrained=True` 从 HuggingFace Hub 下载预训练权重文件（`model.safetensors`）
2. 构建 ConvNeXt V2 Base 网络结构
3. 加载预训练权重到除分类头外的所有层
4. 将最后的全连接层从 1000 类（ImageNet）替换为 2 类（kc / normal）
5. 新的分类头权重随机初始化，其余层保留预训练权重

### 3.3 备选模型

| 模型 | timm 名称 | 参数量 | 特点 |
|------|----------|--------|------|
| ConvNeXt V2 Base | `convnextv2_base` | 88M | 推荐，平衡速度与精度 |
| ConvNeXt V2 Nano | `convnextv2_nano` | 15M | 轻量级，适合低显存环境 |
| MaxViT Tiny | `maxvit_tiny_224` | 30M | 混合局部-全局注意力，推理快 |
| Swin Transformer V2 | `swinv2_base_window16_256` | 87M | 纯 Transformer 架构，输入 256x256 |

### 3.4 模型保存格式

训练过程保存的检查点文件（`best_model.pth`）结构：

```python
{
    'epoch': int,                    # 训练轮次
    'model_state_dict': OrderedDict, # 模型权重
    'optimizer_state_dict': dict,    # 优化器状态
    'val_acc': float,                # 验证集准确率
}
```

## 四、训练流程

### 4.1 训练配置

| 超参数 | 值 | 说明 |
|-------|-----|------|
| 优化器 | AdamW | 带权重衰减的 Adam，适合迁移学习 |
| 初始学习率 | 1e-4 | 预训练模型微调的常用值 |
| 权重衰减 | 0.05 | L2 正则化强度 |
| 学习率调度 | CosineAnnealingLR | 余弦退火，周期 30 轮，最低 1e-6 |
| 损失函数 | CrossEntropyLoss | 交叉熵损失，多分类标准 |
| 训练轮次 | 30 | 默认配置 |
| 批大小 | 16 | 每批 16 张图片 |
| 最佳模型策略 | 验证集最高准确率 | 仅保存 val_acc 最高的检查点 |

### 4.2 训练流程

```
开始
  │
  ├── 加载预训练模型 + 替换分类头
  ├── 加载训练集/验证集/测试集
  │
  ├── for epoch 1 to 30:
  │     │
  │     ├── 训练阶段 (train_epoch)
  │     │     ├── model.train()
  │     │     ├── for batch in train_loader:
  │     │     │     ├── 前向传播 → outputs
  │     │     │     ├── 计算 loss = CrossEntropy(outputs, targets)
  │     │     │     ├── optimizer.zero_grad()
  │     │     │     ├── loss.backward()    # 反向传播
  │     │     │     └── optimizer.step()   # 更新权重
  │     │     └── 记录 train_loss, train_acc
  │     │
  │     ├── 验证阶段 (validate)
  │     │     ├── model.eval()
  │     │     ├── with torch.no_grad():
  │     │     │     └── 遍历验证集，计算 loss 和 acc
  │     │     └── 记录 val_loss, val_acc
  │     │
  │     ├── scheduler.step()  # 更新学习率
  │     │
  │     └── if val_acc > best_acc:
  │           └── 保存 best_model.pth
  │
  ├── 在测试集上评估最佳模型
  │     ├── 加载 best_model.pth
  │     ├── 生成分类报告 (Precision, Recall, F1)
  │     └── 绘制混淆矩阵
  │
  └── 保存训练曲线图
结束
```

### 4.3 学习率调度

使用余弦退火（Cosine Annealing）策略：

```python
scheduler = CosineAnnealingLR(optimizer, T_max=30, eta_min=1e-6)
```

**学习率变化曲线：**

```
lr │
   │╲
   │ ╲
   │  ╲
   │   ╲────────
   │            ╲─────────
   │                      ╲──────────── 1e-6 (eta_min)
   └──────────────────────────────────── epoch
   0                               30
```

训练初期学习率较高（1e-4），加速收敛；后期逐渐衰减到 1e-6，精细调整权重。

### 4.4 评估指标

**训练过程中：**
- **Loss**：CrossEntropyLoss，衡量预测概率分布与真实标签的差距
- **Top-1 Accuracy**：每批预测准确率

**测试集评估：**
- **Precision（精确率）**：TP / (TP + FP)，预测为异常的样本中真正异常的比例
- **Recall（召回率）**：TP / (TP + FN），所有真实异常样本中被正确识别的比例
- **F1-Score**：Precision 和 Recall 的调和平均，综合评估指标
- **混淆矩阵（Confusion Matrix）**：展示各类别的预测分布

## 五、推理预测

### 5.1 推理流程

```
输入图片
  │
  ├── PIL.Image.open() → RGB
  ├── Resize(224) → CenterCrop(224)
  ├── ToTensor() → [3, 224, 224] float32
  ├── Normalize(ImageNet mean/std)
  ├── unsqueeze(0) → [1, 3, 224, 224]  # 添加 batch 维度
  │
  ├── model.eval()
  ├── with torch.no_grad():
  │     └── outputs = model(input_tensor)
  │
  ├── softmax(outputs) → 概率分布 [P(kc), P(normal)]
  ├── argmax → 预测类别
  │
  └── 输出结果
        ├── prediction: 'KC' / 'Normal'
        ├── class_name: '圆锥角膜（异常）' / '正常角膜'
        ├── confidence: 0.9876 (float)
        ├── probabilities: {'圆锥角膜（异常）': 0.9876, '正常角膜': 0.0124}
        └── suggestion: '建议进一步检查' / '符合手术条件'
```

### 5.2 单例模式模型服务

`ModelService` 类使用单例模式，确保整个应用生命周期内模型只加载一次：

```python
class ModelService:
    _instance = None
    _model = None
    
    def __new__(cls):
        if cls._instance is None:
            cls._instance = super().__new__(cls)
        return cls._instance
    
    def __init__(self):
        if self._model is None:
            self._initialize()  # 仅首次初始化时加载模型
```

**优势：**
- 避免重复加载 88M 参数模型（加载耗时 3-5 秒，内存占用 300MB+）
- Web 和 API 服务共享同一个模型实例
- 支持并发请求时无需锁机制（PyTorch 模型推理是线程安全的）

### 5.3 接口支持

**命令行预测（predict.py）：**
- `--image`：单张图片路径
- `--dir`：批量预测文件夹
- `--save`：保存可视化结果图
- `--model`：指定模型权重路径

**批量预测结果导出为 CSV：**

|------|------|------|

| prediction | string | KC / Normal |
